# event-management-mongodb-assignment

# Event Management & Ticketing System (MongoDB Backend)

## Project Overview
This project is a MongoDB backend design for an Event Management and Ticketing System.  
The main purpose of this project is to understand how MongoDB collections, documents, and queries work in a real-life scenario.

In this system:
- Organizers can create events
- Users can browse events
- Users can book or cancel tickets
- Reports can be generated using MongoDB aggregation

This project is created as part of an academic MongoDB assignment.

---

## Technologies Used
- MongoDB
- MongoDB Shell (mongosh)
- Visual Studio Code
- Git & GitHub
